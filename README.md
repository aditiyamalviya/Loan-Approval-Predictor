## 📖 Project Overview

Banks receive thousands of loan applications and must decide whether to approve or reject each one based on applicant data. This project automates that decision using **Machine Learning classification models**, trained on applicant attributes like income, credit history, education, marital status, and property area.

### Objectives
- Clean and preprocess raw loan applicant data
- Visualize patterns and relationships in the data
- Train and compare two ML models (Logistic Regression & Random Forest)
- Predict loan approval for new applicants
- Present everything through an interactive Streamlit dashboard

## 📊 Dataset

This project uses a **synthetic (dummy) dataset** generated by `src/generate_dataset.py`, modeled closely after the well-known public **"Loan Prediction Problem Dataset"** available on Kaggle. It contains 800 applicant records with realistic distributions and intentionally includes missing values so the "Data Cleaning" stage has real work to do.

**Columns:**
| Column | Description |
|---|---|
| Loan_ID | Unique loan application ID |
| Gender | Male / Female |
| Married | Applicant's marital status |
| Dependents | Number of dependents (0, 1, 2, 3+) |
| Education | Graduate / Not Graduate |
| Self_Employed | Yes / No |
| ApplicantIncome | Applicant's monthly income |
| CoapplicantIncome | Co-applicant's monthly income |
| LoanAmount | Loan amount requested (in thousands) |
| Loan_Amount_Term | Term of loan in days |
| Credit_History | 1 = good credit history, 0 = bad |
| Property_Area | Urban / Semiurban / Rural |
| Loan_Status | Y = Approved, N = Rejected (**target column**) |

> You may replace `data/loan_data.csv` with a real dataset from [Kaggle](https://www.kaggle.com/datasets/altruistdelhite04/loan-prediction-problem-dataset) if you prefer — just keep the same column names.

## 🤖 Machine Learning Models

Two classification models were trained and compared:

| Model | Accuracy |
|---|---|
| Logistic Regression | ~96.9% |
| **Random Forest Classifier** | **~97.5% (Best)** |

The **Random Forest Classifier** was selected as the final model and saved to `model/loan_model.pkl`.

Full metrics (Precision, Recall, F1-score, Confusion Matrix) are available in `outputs/model_results.txt` and `outputs/graphs/`.

## 🖥️ How to Run This Project (Step-by-Step)

> A complete beginner-friendly guide (installing Python, VS Code, Git, and GitHub) is provided separately in **`docs/documentation.md`** and in the setup guide shared with this project. Below is the quick version.

### 1. Clone this repository
```bash
git clone https://github.com/YOUR-USERNAME/Loan-Approval-Predictor.git
cd Loan-Approval-Predictor
```

### 2. Create a virtual environment (recommended)
```bash
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # macOS / Linux
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the pipeline in order
```bash
python src/generate_dataset.py     # Step 1: Create the dataset
python src/data_cleaning.py        # Step 2: Clean the data
python src/visualization.py        # Step 3: Generate charts
python src/train_model.py          # Step 4: Train & save the model
python src/predict.py              # Step 5 (optional): Predict via terminal
```

### 5. Launch the interactive dashboard
```bash
streamlit run src/app.py
```
This opens a browser window at `http://localhost:8501` with data overview, charts, and a live prediction form.
