# Screenshots Folder

Add your screenshots here, for example:
- A screenshot of VS Code showing the project running
- A screenshot of the terminal output when running the scripts
- Screenshots of the Streamlit dashboard (all 3 tabs)
- A screenshot of your GitHub repository page

Recommended file names:
- screenshot_1_vscode.png
- screenshot_2_terminal_output.png
- screenshot_3_dashboard_data_tab.png
- screenshot_4_dashboard_charts_tab.png
- screenshot_5_dashboard_prediction_tab.png
- screenshot_6_github_repo.png

See the step-by-step guide provided for exactly how to take and add these.
